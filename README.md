# Internal Webapp – Tailoring Business

A prototype management tool for a tailoring shop.

## Modules
- **Customers** – manage client profiles
- **Measurements** – record body measurements per garment type
- **Orders** – track orders through Pending → In Progress → Completed → Delivered
- **Staff** – manage employees and roles

## Setup

```bash
npm install
npm start
```

Open http://localhost:3000
